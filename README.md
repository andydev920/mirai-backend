

### モジュール

| 项目                       | 説明                 |
|--------------------------|--------------------|
| `mirai-dependencies`     | Maven の依存関係バージョン管理 |
| `mirai-framework`        | Java フレームワークの拡張    |
| `mirai-server`           | 管理者画面とユーザーアプリのサーバー |
| `mirai-module-system`    | システム機能のモジュール       |
| `mirai-module-infra`     | インフラ機能のモジュール       |
| `mirai-module-employees` | 人事機能のモジュール         |
